# Descripción

### Este proyecto nació el día viernes 21 de Noviembre del 2025 a las 22:33 hs con el fín general de brindar un servicio de Tarot Open Source (gratis) a cada usuario consultante de su vida.
La idea del autor y desarrollador, Gastón Ariel Perrone (18 de Diciembre del 1991 - ? , Ciudad Autónoma de Buenos Aires, Buenos Aires, Argentina), a través de este mismo este proyecto personal es demostrar sus habilidades duras (en el transcurso de este documento se desarrollarán) aportadas para el desarrollo del funcionamiento para las finalidades del servicio.
Este mismo está desarrollado y controlado por una API JSON local que se encuentra en este repositorio. Esta API se refiere a las cartas del Tarot y sus descripciones que fueron generadas en su tiempo por ChatGPT (en su gran mayoría) y aprobadas por el mismo autor. Todo su funcionamiento es posible gracias a JavaScript para la selección de cartas y un asistente AI que actúa como tarotista en base a la información del usuario consultante.
